## rdp 协议客户端与服务器握手交互时序图

```
客户端                                         服务器
  |                                               |
  | ---[1] TCP 连接请求 (端口 3389)-------------> |
  | <--[2] TCP 连接建立成功 --------------------- |
  |                                               |
  | ---[3] X.224 连接请求 (Connection Request)--> |
  | <--[4] X.224 连接确认 (Connection Confirm)--- |
  |                                               |
  | ---[5] SSL/TLS 握手初始化 ------------------> |
  | <--[6] SSL/TLS 握手响应 --------------------- |
  |                                               |
  | ---[7] 安全协议协商（Security Exchange）--->  |
  | <--[8] 协议确认 ----------------------------- |
  |                                               |
  | ---[9] CredSSP 认证请求（用户凭据）---------> |
  | <--[10] CredSSP 验证结果 -------------------- |
  |                                               |
  | ---[11] Channel 请求（如 RDPDR、RDPWD 等）--> |
  | <--[12] Channel 响应 ------------------------ |
  |                                               |
  | ---[13] 客户端初始化（客户端信息）--------->  |
  | <--[14] 服务端初始化（显示信息）-----------   |
  |                                               |
  | =======> 建立图形会话，用户开始交互 <=======  |

```

```
RDP over NLA 协议栈流程图
=========================

┌──────────────────────────────┐
│       Application Layer      │
│         (RDP, GDI, etc.)     │
└──────────────┬───────────────┘
               │
┌──────────────▼───────────────┐
│         MCS / GCC            │  ← 会话/通道管理
└──────────────┬───────────────┘
               │
┌──────────────▼───────────────┐
│           X.224              │  ← 初始连接请求/响应（TCON/CC）
└──────────────┬───────────────┘
               │
┌──────────────▼───────────────┐
│       TLS Handshake (SSL)    │  ← 安全通道，建立证书信任
│  - ServerHello/ClientHello   │
│  - Server Certificate        │
│  - Key Exchange              │
└──────────────┬───────────────┘
               │
┌──────────────▼───────────────┐
│     CredSSP (SSPI + SPNEGO)  │  ← 认证层，提供身份验证（NTLM/Kerberos）
│  - Negotiate Protocols       │
│  - NTLM/Kerberos Token       │
│  - Credential Delegation     │
└──────────────┬───────────────┘
               │
┌──────────────▼───────────────┐
│            TPKT              │  ← 数据长度封装
└──────────────┬───────────────┘
               │
┌──────────────▼───────────────┐
│             TCP              │  ← 流传输
└──────────────┬───────────────┘
               │
┌──────────────▼───────────────┐
│              IP              │
└──────────────┬───────────────┘
               │
┌──────────────▼───────────────┐
│       Data Link / Ethernet   │
└──────────────┬───────────────┘
               │
┌──────────────▼───────────────┐
│         Physical Layer       │
└──────────────────────────────┘

```
