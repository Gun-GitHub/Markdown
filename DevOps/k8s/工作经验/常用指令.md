创建命令空间:

```sh
kubectl create namespace <namespace-name>
```

删除命名空间

```shell
kubectl delete namespace <namespace-name>
```

kubectl 指定 config 文件

```shell
# 使用 --kubeconfig 参数（最直接）
kubectl --kubeconfig=/path/to/your/config get nodes
# 使用环境变量 KUBECONFIG
export KUBECONFIG=/path/to/config
kubectl get pods
```
