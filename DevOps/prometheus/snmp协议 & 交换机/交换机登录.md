华为交换机链接方式:

```sh
telnet 192.168.1.254
username: adminuser
passwd: funsine@123! # 如果不行就 funsine@123 这是初始密码,登录以后会让你改密码改密码就添加一个 !


telnet 10.10.10.18
username: adminuser
passwd: FUsl2023!.
```

华为交换机开启 snmp 服务:

```sh
# 进入系统模式
system-view
# 显示当前配置
display current-configuration 
# 添加一个 snmp-agent 设置你自己的密码
snmp-agent community read cipher <yourpassword>
# 退出系统模式
quit
# 保存所有配置
save
```
