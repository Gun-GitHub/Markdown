### 刷新或者缓存用户指令

```sh
sss_cache --users
```

### 刷新ldap所有配置缓存

```sh
sss_cache -E
```

### 刷新或者缓存用户的方法

```sh
systemctl restart sssd
```

### 同步ldap用户的账户密码的指令

```sh
authconfig --enablesssd --enablesssdauth --enablelocauthorize --update
```

### 这条指令的作用是使用authselect工具选择SSSD（System Security Services Daemon）作为系统的身份验证和授权机制，并强制应用这个选择。
