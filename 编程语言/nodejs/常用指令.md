## npm 常用指令

如果你想把 **npm 源指定为清华大学镜像**，可以用下面命令：

```bash
npm config set registry https://mirrors.tuna.tsinghua.edu.cn/npm/
```

然后验证是否成功：

```bash
npm config get registry
```

如果输出：

```
https://mirrors.tuna.tsinghua.edu.cn/npm/
```

说明已经切换成功。 ([墨天轮](https://www.modb.pro/db/1935510312669687808?utm_source=chatgpt.com))

## 切回官方源

如果以后需要发布包：

```bash
npm config set registry https://registry.npmjs.org/
```

## 进阶（推荐）

如果你经常切换源，可以安装 **nrm**：

```bash
npm install -g nrm
nrm ls
nrm use taobao
```

可以一键切换不同 npm 源。 ([源码学社](https://code.itptg.com/article/24?utm_source=chatgpt.com))

如果你 **只想临时使用清华源（只对当前命令生效）**，可以在安装时指定 `registry` 参数：

```bash
npm install <package-name> --registry=https://mirrors.tuna.tsinghua.edu.cn/npm/
```

例如：

```bash
npm install express --registry=https://mirrors.tuna.tsinghua.edu.cn/npm/
```

这样 **不会修改全局 npm 配置**，只对这一条命令有效。

------

### npx 也可以临时指定

如果用 `npx`：

```bash
npx --registry=https://mirrors.tuna.tsinghua.edu.cn/npm/ create-react-app myapp
```

------

### 另一种写法（环境变量）

也可以：

```bash
npm_config_registry=https://mirrors.tuna.tsinghua.edu.cn/npm/ npm install express
```

------

💡 **建议记住最常用这一条就行：**

```bash
npm install 包名 --registry=https://mirrors.tuna.tsinghua.edu.cn/npm/
```
